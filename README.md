# Quip to PDF Slack app
This Slack app provides a slash command (/quiptopdf) to export Quip document to PDF.

Need to set the following environment variables.
* SLACK_SIGNING_SECRET
* SLACK_BOT_TOKEN
* QUIP_ACCESS_TOKEN
